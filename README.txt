Youtube Formatter
===================

What is this?
-------------

Youtube Formatter module include a format option for the plain text fields to
include the Youtube video embed giving the video ID.

Install
------------

Youtube Formatter is installed like any other Drupal 8 module: place the module
folder into the modules/ directory in your Drupal 8 installation.

Within the Drupal, enable the module in Admin menu > Extend.

How to use it
--------------

Just select the "Youtube Formatter" format option in any Text (plain) field on
the "Manage Display" tab of the Content type edition.

Be sure to include a valid Yotube ID video as a value of the formatted field.
